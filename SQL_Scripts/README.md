# 📂 Módulos de Procesamiento SQL

Este directorio contiene el flujo completo de ingeniería y análisis de datos.

1. **01_data_cleaning.sql**: Normalización de fechas y deduplicación.
2. **02_data_modeling.sql**: Implementación de PKs, FKs y Modelo Estrella.
3. **03_exploratory_data_analysis.sql**: Métricas base y validación de volumen.
4. **04_business_performance_insights.sql**: Análisis de ventas, canales y estacionalidad.
5. **05_customer_store_behavior.sql**: Segmentación y comportamiento de puntos de venta.
6. **06_market_basket_engine.sql**: Lógica de recomendación de productos.
